课程简介：


课程采用FFMpeg作视频解码，采用OpenGL实现视频的渲染，

从RGB数据的绘制，到直接渲染YUV数据，实现三路YUV数据单纹理的输入，

提升系统性能，接着采用PBO方式优化内存到显存的传输，实现DMA传输，

解放CPU,将解码和渲染分离到不同的线程中，极大的提升了系统的性能。

最后介绍视频特效的实现。
